# Security Policy

The security of ProSite and its users is taken seriously. This document explains how to report security vulnerabilities responsibly.

---

## Reporting a Vulnerability

If you discover a security issue in ProSite, please report it privately instead of opening a public issue.

You can report vulnerabilities through:

GitHub: https://github.com/r3xden  
Discord: https://discord.gg/6HKvcewn8w  

Provide as much detail as possible, including:
- Description of the issue
- Steps to reproduce
- Potential impact
- Screenshots or proof if applicable

---

## Response Policy

- All valid security reports will be reviewed promptly.
- If confirmed, a fix will be developed and released as soon as possible.
- Credit may be given to the reporter (if desired).

---

## Scope

This policy applies to:
- Website source code
- JavaScript functionality
- UI interaction logic
- Theme & storage logic
- Any deployed version of ProSite

---

Thank you for helping keep ProSite safe and secure.
