# ProSite Branding & Usage Guidelines

These guidelines define how the ProSite name, identity, and branding may be used.

------------------------------------------------------------

## Ownership

ProSite is an open-source project created and maintained by:

Anmol Singhal (Rexden)

While the code is open-source, the ProSite name, brand identity, and associated logo remain protected.

------------------------------------------------------------

## Allowed Usage

You MAY:

- Fork ProSite and modify it for personal or commercial use
- Use ProSite as a base for your own website
- Customize layout, styling, and structure
- Attribute ProSite in your project credits
- Use phrases such as:
  - "Built with ProSite"
  - "Based on ProSite"
  - "Powered by ProSite"

------------------------------------------------------------

## Restricted Usage

You MAY NOT:

- Rename your fork or derivative project as "ProSite"
- Claim that you created the original ProSite project
- Sell ProSite as a standalone commercial product without attribution
- Misrepresent ProSite as your own proprietary framework or software
- Use the ProSite name in a way that implies official endorsement without permission

------------------------------------------------------------

## Logo & Visual Identity

If ProSite branding assets (logo, banners, icons) are provided:

- You may use them only for attribution or reference
- You may not modify the logo to create a competing brand
- You may not remove original creator credit from distributed branding

------------------------------------------------------------

## Attribution Requirement

If you redistribute ProSite or create a derivative project, you must include:

- A visible credit to:
  "ProSite by Anmol Singhal (Rexden)"
- A link to the original GitHub repository

------------------------------------------------------------

## Enforcement

Misuse of ProSite branding may result in:

- A takedown request
- Removal of attribution privileges
- Possible legal escalation for severe misuse

------------------------------------------------------------

## Contact

For branding permissions or special usage requests, contact:

onlyanmolbusiness@gmail.com

------------------------------------------------------------

ProSite branding exists to protect the integrity of the project while allowing open-source freedom.
